数组

数据类型 数组名[]

​ java 定义数组时，不会为数组分配存储空间——数组声明不指名长度

​ 数组定义制建立了数组的引用，需要用new分配内存空间。

char s[]; s = new char [5];
int temp = new int [10];//在定义数组的同时为之分配内存空间
int s1[] = {1,2,3}//内存分配，初始化工作放在一条语句

多维数组类似

java.util.Arrays常用方法： sort，equals（两个数组是否相等），

copyOf（type[] original, int length): 将原数组复制为新数组，length为新数组的长度

强引用：

一个对象具有强引用，他就不会被垃圾回收器回收

内存不足时：JVM也不会回收它，而是抛出OutOfMemoryError

终端强引用和某个对象之间的关联，可将引用赋值为null。

eg：

Object strongRef = new Object( );
