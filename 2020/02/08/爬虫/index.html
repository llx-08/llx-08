概述

基本原理其实就是对网站发出一个请求，获取网站的信息后对其进行分析，进行下一步爬取然后blablabla

当然为了保证编写的爬虫都是合法的，还是建议在爬取网站之前先查看该网站的robots.txt，直接在网站url后面加上/robots.txt 就可以了，看看那些让爬那些不让爬。写的时候也不要爬太快把别人服务器爬崩了，用python写的话加上点 time.sleep(1)啥的，当然这些都是后话

REQUESTS

requests库，没装的用pip install requests装一下

import requests
url = input("输入需要爬取的网址：")
resp = requests.get(url)
print(resp)
print(resp.text)

如果你的网络没有问题的话，并且爬取的网站没有反爬机制的话，那么首先print出来的应该是

好像只要方括号里面的是2开头的就都没有问题，3开头或者是4开头的就没有获取到页面了。

然后下面print的是网页的html代码，其实这就是完成了一个简单的爬虫。

BeautifulSoup

这个库还蛮厉害的，可以让爬出来的东西显示的更简洁。

首先要做的就是调库

from bs4 import BeautifulSoup
soup = BeautifulSoup(html, 'lxml') #html是获取的网页代码？不知道怎么形容
#反正比较主流的要么是urllib.urlopen('url').text，要么是requests.get('url').text
#lxml是解释器，也可以是html.parser等等
#这里的soup就是一个BeautifulSoup对象，可以调用下面的函数更清晰的认识网页

常用的函数:

prittify( ):将网页的html代码按照标准的缩进格式输出

find()：找到第一个符合查找要求的，参数和下面的函数一样

findAll(name , attrs , recursive , text , **kwargs)：找到所有符合要求的，好像这个函数和find_all( )没区别。

几个参数分别代表：查找的标签名，像是’img’, ‘p’；查找的属性名，’class’, ‘id’；关键词 ； 需要查找的文本。

基本重要的知识就这些了，别的东西用到了再说吧，但用这种方法手动创建的爬虫只要爬取的网站稍微有一点反爬措施就gg，即使使用headers或者代理服务器什么的也不太行（亲测，我把headers里所有东西都加上，包括cookies，referr那些，该不行还是不行）。

在这里给一点推荐，一个是Scrapy框架，另外一个是selenium库。这两个东西至少我目前写的爬虫没有遇到任何阻拦，马上要看网课了，后面再接着写。
